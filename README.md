# Mantle Turing Test Hackathon Workspace

Research and product-planning workspace for a Mantle-native, policy-aware AI
treasury assistant.

## Start Here

1. Read [Project Knowledge](knowledge/00-project-knowledge.md).
2. Review the [recommended product](knowledge/product/recommended-concept.md).
3. Use the [contract registry](knowledge/reference/contracts.md) before integration.
4. Check the [hackathon requirements](knowledge/hackathon/requirements.md) before
   submission.
