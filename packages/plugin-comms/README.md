# @nebula/plugin-comms

A2A messaging + ERC-8183 marketplace plugin for **nebula**. Ships 11 brain limbs for `agent.message`, `agent.contacts`, `agent.history`, plus 9 `market.*` limbs for the autonomous job lifecycle.

Encrypted A2A via ECIES on the NebulaInbox singleton (`0xcd92844cc0ec6Be0607B330D4BaCC707339f2589` on 0G mainnet). Marketplace via NebulaMarket (`0x3ebD21f5dd67acDeF199fACF28388627212bA2aB`).

## Install

Auto-installed when you `bun add -g @nebula/cli`. Or directly: `bun add @nebula/plugin-comms`.

Requires [bun](https://bun.sh) ≥ 1.1.

See the [root README](https://github.com/rstfulzz/nebula#readme) for the full plugin and tool surface.
